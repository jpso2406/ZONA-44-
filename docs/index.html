Manual de Instalación

  1. Descargar el instalador: Visite la página oficial y descargue el instalador compatible con su sistema operativo.
  2. Ejecutar el instalador: Abra el archivo descargado y siga las instrucciones del asistente de instalación.
  3. Configurar opciones: Seleccione las opciones deseadas durante la instalación (ruta, componentes, etc.).
  4. Completar instalación: Espere a que el proceso finalice y haga clic en “Finalizar”.
  5. Verificar instalación: Abra la aplicación para confirmar que se ha instalado correctamente.
